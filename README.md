# Web-site Fake Airbnb construido com JavaScript.

» Projeto acadêmico feito com intuito de ser uma réplica do Airbnb com algumas funcionalidades do mesmo.

» O sistema em sua base foi construido em Javascript, React, Vite, Tailwindcss, Express, Axios, Moongose, MongoDB, AWS-Sdk, Cors, JWT entre outros.

» Funcionalidades como formulário completo com perfil, Login e criação de post places, check-in, check-out, log de reservas e uma navegação fluída em layouts de imagens hospedada na Aws-s3, com banco de dados mongodb a parte.

» Deploy feito no servidor Render.

Link: https://website-airbnb-react.onrender.com
